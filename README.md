# Reproducible Research: Peer Assessment 1

## Project Overview
This project analyzes data from a personal activity monitoring device 
(e.g. Fitbit, Nike Fuelband). The device collected data at 5-minute 
intervals throughout the day during **October and November 2012**.

## Dataset
- **File:** `activity.csv`
- **Variables:** `steps`, `date`, `interval`
- **Observations:** 17,568

## Assignment Questions Answered
1. What is the mean total number of steps taken per day?
2. What is the average daily activity pattern?
3. How should missing values be imputed, and what is the impact?
4. Are there differences in activity patterns between weekdays and weekends?
